# Database Nexus Assistant Skill

Skill Markdown para orientar el uso de Drop Nexus: importacion de bases, preparacion de replicas, modos de escritura y solucion de fallos.

## Uso

- Usa `SKILL.md` como referencia del asistente en herramientas compatibles con skills Markdown.
- En la app web, la misma guia vive en `frontend/src/skills/database-nexus-assistant.md`.
- No pegues contrasenas, tokens ni URLs privadas de bases de datos dentro del chat.
